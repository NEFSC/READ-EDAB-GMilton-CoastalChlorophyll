# coastal-chlorophyll
Code repository for collecting and organizing in-situ North American coastal chlorophyll 

## Description
Satellites and other algorithms have a difficult time accuratly detecting chlorophyll in coastal regions do to optically complex waters. In situ data can be used as ground truth and to train models to better detect chlorophyll. This repository includes Python Spyder codes for organizing chlorophyll data from a wide range of publically avaoiable databases.

Code folder holds code for concatinating and organizing raw files into xlsx files.
